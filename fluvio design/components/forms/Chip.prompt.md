Pill filter/tab used inside cards. Active is a solid violet fill; inactive is mist text that darkens on hover.

```jsx
<Chip active>Todos</Chip>
<Chip>Pendientes</Chip>
<Chip>Confirmadas</Chip>
```

Render a row of Chips as a filter group above a table. For a single bordered control, use the `.seg` segmented pattern instead.
